
# Todo App

A simple Todo App inspired by Microsoft To Do.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using Local Storage
- Simple and responsive user interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

```
todo-app/
│── index.html
│── style.css
│── script.js
└── README.md
```

## How to Run

### Method 1

Open `index.html` directly in any web browser.

### Method 2

Run a local server:

```bash
python3 -m http.server 8000
```

Then open:

```
http://localhost:8000
```

## Inspiration

This project replicates the core functionality of **Microsoft To Do**, which has more than 100K+ downloads.
